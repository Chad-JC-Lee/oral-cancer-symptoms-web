# Assets 目錄

這個目錄包含應用程式的圖標和啟動畫面圖片。

## 需要的檔案

- `icon.png` - 應用程式圖標 (1024x1024)
- `splash.png` - 啟動畫面 (1242x2436)
- `adaptive-icon.png` - Android 自適應圖標 (1024x1024)
- `favicon.png` - Web 圖標 (48x48)

## 圖標設計建議

- 使用醫療相關的圖標設計
- 色彩溫和，適合病人使用
- 圖標清晰，易於識別
- 符合無障礙設計原則

## 啟動畫面設計建議

- 簡潔的設計
- 包含應用程式名稱
- 使用溫和的背景色
- 避免過於複雜的圖案
